My perfect terminal setup in 3 simple steps

Download and install zsh

automatic script: it is under development but you van try it : automatic script You can see the development stage there.

Download and install zsh

Why zsh? for me its the completion features and the customizability that it offers. (it is not as bloated as bash xD)

Depending on your distro you may want to do: sudo pacman -S zsh or sudo apt install zsh then you want to set it to default, to view which is your default shell, type: echo $SHELL if you have never changed it maybe it prints bash to swap bash for zsh, do: chsh -s $(whereis zsh)

You may have to configure it, if you dont want to get into a lot of detail I would suggest to grab a good config file like the Luke Smith zsh config file its the one I used as a base.

Note that some of the features may not work with your setup. I also added History configuration to my config. But the important part that I like about this one is the compinit part and the zsh-syntax-plugin (at the end, you have to install this manually, its very easy :)

Install and configure st

This is the terminal emulator, go ahead to suckless website to learn more, this a program that you compile for yourself, but its pretty fun!!

To do this, you download the st 0.8.2 (at the moment) in the st.suckless website, it can be found right here: st scroll down to the Download part and click the st link. You will then get a tar.gz, untar it. Then you will see a folder named st0.8.2 or whatever version number, cd into it and just issue a sudo make install command. You will have now installed st, try it by launching it with another terminal or via the menu or whatever method you use in your distro :)

Ricing

This is the part where we actually get the fonts and set the prompt!

First install the font : Nerd fonts I found the Iosevka Nerd Font to work well, just hit the download button. You may want to refer to other place to install the downloaded font as there may be a better way to do it than the one I am about to explain: copy the file you just downloaded to .local/share/fonts/misc/ and then issue the command : fc-cache . You may want to make sure that the font is there , you do this with the : fc-cache | grep "Nerd" command (for example xD)

Its time to change the fonts on the config.h file (under the st directory that should be somewhere in your computer ) . Replace the tow font lines for the following: 

 
static char *font = "Iosevka Nerd Font:size=14";
This should setup the fonts (make sure you have the inconsolata one too, but really any other can go there as well as it doesnt interfere with the icons). The idea is to put a first font for the text itself that cant support the icons, and so the second one will come into place, we do this to be able to tweak the size so the icons look good. Note that you have to recompile st, but this is trivial if you installed from luke , just type sudo make -i in the st diectory :)

We are finally ready to change the prompt, go ahead to the .zshrc file and replace the PS1 for one with the icons you like from the Nerd Fonts cheatsheet !
